Hi,

After too much time without a new release, today we released version
0.0.4 of Buxon sioc:Forum browser [1]. It's available to download in
BerliOS [2].

The ChangeLog of this new version could summarized in:

  - PingTheSemanticWeb.com support added
  - Updated to the latest version of RDFLib
  - Deleted dependency on SWAML core
  - Redesigned fetcher and cache
  - Fixed some minor bugs

Our intentions were wait some time more until finish some other
features. But a bug (fixed in svn many months ago) has been reported to
Debian [3], so it's necessary to publish ASAP a version with these
fixes.

Sorry Nacho for the problems.

Cheers,
