Hi,

these days we have been working very hard to improve the project. Now
SWAML also export XHTML+RDFa+GRDDL, integrated with RDF/XML using
content negotiation over a new URIs scheme (see an example [1]). Sorry
to announce it before in other forums [2].

All these new feature are explained in a new revision of our last paper
[3] that has been accepted in the IADIS International Journal on
WWW/Internet [4] (but still pending to appear).

In the next days we'll stabilise all the changes, and it will be time to
publish a new release.

And that's all, folks.
