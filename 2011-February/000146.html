Hi,

after centuries without activity on the project, this weekend I was
working on a hacking prototype known as LinkedMarkMail [1].

Based on this unofficial notes [2], I've developed a small tool for
exposing as Linked Data the mailing lists' archives indexed by
MarkMail [3]. Although I started the development of a new SWAML API
(what I actually wanted to do some time ago...), the generated RDF
should be fully compatible with other tools (SWAML; Buzon, and so on).

I still didn't announced it in more popular forums, because I'm not
sure about the legitimacy for using that API. But your feedback would
be welcomed while I get an answer about this issue.

Thanks you very much.
