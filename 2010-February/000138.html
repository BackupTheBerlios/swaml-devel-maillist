Hello,

I've uploaded version 0.1 of Siocwave[1] -- a new SIOC site browser,
akin to Buxon. It's a desktop tool based on Python and wxWidgets, so
should be runnable on most platforms.

The goal of Siocwave is to provide a view of SIOC data on a logical
level, as opposed to manually inspecting individual RDF statements. You
feed data into Siocwave, and it displays a hierarchy of
containers/items/replies that it can see, plus provides you with links
to fetch more related data (e.g. via rdfs:seeAlso). I hope that it may
be useful in exploring and debugging SIOC sites.

For more information, please see the Google Code project at [1].

Feedback will be highly appreciated.

[1] http://code.google.com/p/siocwave/
