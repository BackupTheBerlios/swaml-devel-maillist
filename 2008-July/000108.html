Hi folks,

it's my fault, I know it. During these months we introduced many new
features in the project, but we didn't publish it as a release. I think
this is a good moment, for many reasons (not for time, as usual):

- Many (and important) features [1] are missing for people (svn is not
for users).

- The policy of versions is really bad. I need to fix it, not only to
reflect the real work made in the project, but also to be able to
publish small revisions. So this release will be 0.1.0.

- We need a target where start to work on an old promise [2] with Linked
Data community. I'm not sure if I'll have time to do it. But I'm sure
that SWAML needs to be able to manage efficiently updates of the
archives [3].

- ﻿On next Sunday Debian will freeze the packages that will go in the
next stable release of this popular distribution, so we should arrive in
time.

For all these reasons, I want to publish a new release asap. The actual
code in the svn (revision #1120) is stable enough. But if you have ﻿any
comment/bug, please sent it to me. Anyways tomorrow I'll check it with
detail, running some heavy tasks on it. And if everything is right, by
the end of the day I'll prepare a proper release.

I think that's all... Cheers.
